* **진행 기간** : 2022. 02. 11 ~ 2022. 02. 17
* **사용 스킬** : `Pandas` `Numpy` `Matplotlib` `Seaborn` `Scipy`

## *III. 데이터 시각화 & 분석*
### 어떤 시장을 집중적으로 공략해야 할 것인가?
현재 데이터상으로 크게 북미, 유럽, 일본, 기타 지역으로 나뉘어져 있는데,  
각 지역별로 선호하는 게임의 종류가 다를 것이라 생각해 만약 특정 시장을 집중 공략해야한다면 어느 지역을 타겟으로 잡아야할지 확인해보았다

**북미 시장을 공략**하는 것이 효과적일 것이다.

## *IV. 가설 생성 & 검정*
최종적으로 결정하기 전 타겟 시장인 북미지역, 그리고 전세계적으로 더 많이 팔리는지 확인하기 위해  
두 가지 사항에 대해 2-Sample T-test를 통해 가설검정을 해보았다
### 가설 1. 북미 지역에서 다른 게임에 비해 더 잘 팔리는가?
귀무가설(H0) : X360 기반 슈팅 게임의 북미지역 평균 출고량은 다른 게임의 북미 지역 평균 출고량과 차이가 없다.  
대립가설(H1) : X360 기반 슈팅 게임의 북미지역 평균 출고량은 다른 게임의 북미 지역 평균 출고량보다 높다.
### 가설 2. 전 세계에서 다른 게임에 비해 더 잘 팔리는가?
귀무가설(H0) : X360 기반 슈팅 게임의 전세계 평균 출고량은 다른 게임의 전세계 평균 출고량과 차이가 없다.  
대립가설(H1) : X360 기반 슈팅 게임의 전세계 평균 출고량은 다른 게임의 전세계 지역 평균 출고량보다 높다.
### 검정 결과

|가설|T-statistic|P-Value|H0 기각 여부|해석|
|:---:|:---:|:---:|:---:|---|
|1|7.8640|0|기각|X360 기반 슈팅 게임의 북미지역 평균 출고량이 더 높다|
|2|10.6082|0|기각|X360 기반 슈팅 게임의 전세계 평균 출고량이 더 높다|
